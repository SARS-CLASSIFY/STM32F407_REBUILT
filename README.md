# **stm32f407 TIM_CAP_REBUILT**

- ## 校赛代码

- 引脚配置

  ```
  /* ----------------PA0 -> 输入捕获-----------------------
  				   PB1 -> PWM输出
  				   PE0 -> OLED SCL
  				   PE1 -> OLED SDA
  				   PA4 -> DAC1
  				   PA5 -> DAC2
  				   PE0 -> OLED1 SCL
  				   PE1 -> OLED1 SDA
  -------------------------------------------------------*/
  ```

- 定时器采用PWM模式输出捕获测频，可测占空比 TIM5定时器使能

- DAC PWM输出波形，PWM到100K接负载消除振铃，DAC较高频率不可接负载

- 矩阵键盘实现交互操控

- 自写简陋的状态机函数STAGE来完成整个系统操控
